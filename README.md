# 🚀 Sci-Fi Flappy Bird Clone

A high-octane, sci-fi reimagining of the classic "Flappy Bird" mechanics. This project features a sleek spaceship aesthetic, dynamic difficulty scaling, and a reactive plexus particle system that creates an immersive, neon-drenched atmosphere.

![License](https://img.shields.io/badge/license-MIT-blue)
![Status](https://img.shields.io/badge/status-active-green)

## 🌌 Overview

Experience the void in this sci-fi themed arcade game. Instead of a bird, you pilot a sleek spaceship through a shifting nebula of light and dark purple pillars. The environment reacts to your movement, with particles tearing away or being attracted by your ship's engine.

## ✨ Features

*   **Dynamic Difficulty**: Obstacle speed increases gradually over time.
*   **Smart Spawning**: Uses a dynamic spawn rate calculation to ensure that as the game speeds up, the physical distance between obstacles remains consistent and playable.
*   **Plexus Particle System**: A reactive background of particles that creates "plexus" connections, pulsating with a breathing alpha effect for a deep sci-fi feel.
*   **Neon Visuals**: Features high-contrast visuals, including glowing pipe outlines, horizontal sci-fi patterns, and engine fire effects.
*   **Responsive Design**: The game automatically scales to fill your browser window while maintaining the correct aspect ratio and gameplay logic.

## 🛠️ Technologies Used

*   **HTML5 Canvas**: For high-performance 2D rendering.
*   **JavaScript (ES6+)**: Core game engine, physics, and state management.
*   **Tailwind CSS**: For clean, responsive UI elements like the scoreboard.
*   **LocalStorage API**: To persist your High Score across sessions.
